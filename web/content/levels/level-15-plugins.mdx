## 플러그인이란?

스킬, 에이전트, Hook, MCP 서버, LSP 서버를 하나로 묶은 **패키지**.

## 관리

```bash
# VS Code에서
/plugins           # 플러그인 매니저

# CLI에서
claude plugin list
claude plugin install <url-or-name>
```

## 플러그인 소스

- GitHub 리포지토리
- Git URL
- NPM 패키지
- 공식 Anthropic 마켓플레이스

## 설정 예시

```json
{
  "enabledPlugins": {
    "oh-my-claudecode@omc": true,
    "superpowers@claude-plugins-official": true
  }
}
```

> `oh-my-claudecode`: 멀티에이전트 오케스트레이션 플러그인

> `superpowers`: 워크플로 스킬 모음 플러그인
