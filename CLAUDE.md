# Claude Code 학습 프로젝트

이 프로젝트는 Claude Code의 기능을 학습하고 실험하기 위한 프로젝트입니다.

## 프로젝트 규칙

- 한국어로 응답해주세요
- 코드 예시에는 항상 주석을 달아주세요
- 실험 결과는 `/experiments` 디렉토리에 저장해주세요
- 단계별 학습이 완료되면 `/experiments/claude-code-study-guide.md` 를 최신화 해주세요

## 기술 스택

- 학습용 프로젝트 (특정 스택 없음)
- 실험 파일: JavaScript, Python, Shell Script 등 다양하게 사용
